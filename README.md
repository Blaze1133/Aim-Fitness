# AIM FITNESS

Production-ready Next.js implementation of the AIM FITNESS gym website.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Inquiry Form

The contact form posts to `/api/inquiry`. Set `INQUIRY_WEBHOOK_URL` in production to forward submissions to a CRM, automation tool, or email webhook.

## Deploy

This app is ready for Vercel or any Node-compatible Next.js host:

```bash
npm install
npm run build
npm run start
```
